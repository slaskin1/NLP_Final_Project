# Welcome to our Natural Language Processing final project!

This repository contains the .csv files and their iterations, the code we used to clean and analyze the data, the presentation, and final report with findings.

# File Directory

| **File Name**  | **Description** |
| ------------- | ------------- |
| urgent_care_reviews_CA.csv  | Original scraped data  |
| urgent_care_reviews_CA_fully_preprocessed.csv  | Contains review-level text at each preprocessing stage, from raw reviews to fully cleaned and normalized text, supporting downstream NLP tasks.  |
| urgent_care_reviews_SA_summary.csv | Review-level dataset containing raw and cleaned text, star ratings, and sentiment scores (polarity, subjectivity, and NLTK compound) for urgent-care reviews, used to evaluate and compare textual sentiment with user ratings. |
| review_text_statistics_summary.csv | 
| urgentCare_RatingTrends.ipynb | Colab file of trends |
| TopicModelling.ipynb | Topic modeling colab |
| DataCollectionPreprocessingSA.ipynb | Preprocessing pipeline |
| Supervised_Learning.ipynb | Colab with supervised learning code |
| lda_3topics_top_words.csv | Modeling shows 3 topics is most accurate |

*We also have the .pptx presentation and .docx which are included in the final deliverables for the project.*
